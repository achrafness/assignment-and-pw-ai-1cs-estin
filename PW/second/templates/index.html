Timetable Scheduler for 1CS Students - Semester 2

This application uses Constraint Satisfaction Problem (CSP) techniques to generate a feasible timetable for 1CS students.

Constraints:

Hard Constraints:

  • C1: The week consists of five days: Sunday, Monday, Tuesday, Wednesday, and Thursday.
  • C2: Each day has five work slots, except Tuesday which has only three in the morning.
  • C3: Four or five successive slots of work are not accepted (max three successive slots).
  • C4: Lectures of the same course should not be scheduled in the same slot.
  • C5: Different courses for the same group should have different slot allocations.
